# CookMate Frontend

A React-based frontend application for managing recipes, meal plans, and shopping lists.

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Tech Stack

- React 18
- Vite
- React Router DOM
- Axios
- Tailwind CSS
